### Exploring-Random-User-Data-with-Python-Pandas-and-CSV
## Robert Morris University - Advanced Python for Data Analytics - Coursework 
This homework consists of three Python programs that practice working with APIs, pandas, and CSV files using the Random User API. 

# Program 1: 
In this program, I request full information for 25 people, extract specific fields (such as name, email, location, nationality, and picture URL), convert the data into a pandas DataFrame, save it to a CSV file. Then use crosstabs to show counts of people by gender and nationality. 

## Program 2:
In this program, I repeat the process but use API query parameters to limit the response to only the needed fields and to users of a single gender from the US, then save them to a different CSV file. 

## Program 3:
In this program, I load one of the CSV files into pandas and print the resulting DataFrame.
